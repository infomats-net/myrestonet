# **App Name**: DineFlow Global

## Core Features:

- Multi-tenant Restaurant Admin Dashboard: A secure dashboard for restaurant administrators to manage their specific menu items, customize local SEO settings, manage staff roles, and process customer orders in real-time.
- Global Super Admin Portal: A centralized dashboard for platform super administrators to manage subscriptions, oversee platform-wide configurations, create new restaurant admin accounts with assigned credentials, and impersonate restaurant admin roles for support.
- Customer-Facing Ordering Web App: A fast, responsive, and mobile-first web application enabling end customers to browse localized menus and place multi-currency orders seamlessly.
- AI-Powered Sales Analytics: Provides restaurant admins with insightful sales analytics by using AI as a tool to identify trends, popular items, and peak ordering times from transactional data.
- Secure Role-Based Access Control (RBAC): Implements Firebase Authentication with Custom Claims to enforce granular permissions and data isolation, ensuring restaurant admins can only access their restaurant's data while super admins have global access.
- Real-time Menu and Inventory Management: Enables restaurant admins to update their menu items and manage inventory levels in real-time, with immediate reflection on the customer-facing application.
- Multi-Currency Payment Processing: Integrates Stripe Connect to securely handle multi-currency payment transactions, supporting various global currencies for customer orders.

## Style Guidelines:

- Color scheme: Light and professional, promoting clarity and efficiency. Primary elements utilize a sophisticated deep blue (#42668A) to convey trust and authority, while the background is a subtle, light blue-grey (#F0F2F4) to ensure content stands out and loads quickly. Key actions and highlights are accented with a fresh, professional green (#53C683) for visual impact and positive reinforcement.
- Headline and body text font: 'Inter' (sans-serif), chosen for its modern, neutral aesthetic and high legibility across all screen sizes, aligning with the app's performance and responsive design goals.
- Utilize a clean, minimalist line-art icon set to maintain a lightweight design that contributes to sub-1-second page loads. Icons should be clear and intuitive, supporting universal understanding across different languages and cultures.
- A mobile-first, adaptive grid layout that dynamically adjusts content for optimal viewing on any device. Dashboards feature card-based design with clear information hierarchy to ensure quick data consumption and efficient interaction, minimizing user effort.
- Subtle, performant micro-interactions and transitions to provide fluid user feedback without impacting loading times. Animations will guide the user through workflows and enhance responsiveness, such as smooth content reveals or state changes on interactive elements.
